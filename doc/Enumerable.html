Constructor

Enumerable ()

Enumerable is an API that can be mixin other objects. All method of Enumerable use `each` method. But Enumerable don't have `each` method. Therefore, object.prototype to mixin must have `each` method.

Instance Methods 

  all() -> bool
  all(callback) -> bool

Return true when all object == true or all callback returns true.

[PARAM] callback: 
  Enumerator(function(y){
    y(true); y('1'); y(1);
  }).all();
  //=> true
  Enumerator(function(y){
    y(10); y(15); y(32);
  }).all(function(i){
    return i % 5 === 0;
  });
  //=> false
